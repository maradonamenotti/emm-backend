# Sistema de Analíticos - Escuela Maradona Menotti

## Descripción
Módulo de analíticos para el seguimiento académico de alumnos.
Gestiona calificaciones, progreso por módulo y estadísticas por comisión.

## Desarrollado por
Manuela Castro Arellano
Área de Gestión y Sistemas - Escuela Maradona Menotti
Inicio del proyecto: 2025

## Tecnologías
- Backend: NestJS + TypeScript
- Base de datos: PostgreSQL
- Deploy: Railway (en migración a Hostinger)

## Documentación de la API
Disponible en `/api/docs` una vez deployado.

## Estructura
- `/src/alumnos` → gestión de alumnos
- `/src/notas` → gestión de calificaciones
- `/src/analytics` → lógica de analíticos
